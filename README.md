## Обзор

**Dating Bot** — Telegram-бот для знакомств с микросервисной архитектурой. Пользователи создают анкеты, просматривают профили других пользователей, ставят лайки, получают мэтчи и переписываются. Система ранжирования формирует персонализированную ленту анкет на основе трёхуровневого рейтинга.

### Ключевые бизнес-процессы

- **Регистрация** — пользователь запускает бота командой `/start`, система создаёт аккаунт по `telegram_id`
- **Создание анкеты** — пошаговое заполнение профиля через диалог с ботом (имя, возраст, пол, город, интересы, фото, описание)
- **Просмотр анкет** — бот показывает анкеты из персонализированной очереди, отсортированной по комбинированному рейтингу
- **Свайпы** — пользователь ставит лайк или пропускает анкету
- **Мэтч** — при взаимном лайке оба пользователя получают уведомление
- **Чат** — обмен сообщениями после мэтча (через бота или в личной переписке)
